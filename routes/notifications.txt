# Notification-Related Routes

## Admin API Endpoints (Bell Dropdown)
GET    /admin/notifications-api/count          # Get new notification count
GET    /admin/notifications-api/               # Get notification list (max 50)
PATCH  /admin/notifications-api/{id}           # Mark single notification as seen
PATCH  /admin/notifications-api/mark-all-seen  # Mark all notifications as seen

## Middleware & Guards
- Admin routes: web middleware + admin auth
- API routes: tenant context middleware
- CSRF: Required for PATCH operations
- CORS: Handled by framework
